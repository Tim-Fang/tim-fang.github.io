花10分钟，搭建你自己的博客

十一月 29, 2020 1711

https://waterblock79.github.io/2020/07/13/How%20to%20bulid%20Hexo/

环境配置

在搭建Hexo之前，你需要确保有：

  • Node.js

    在 Node.js Download 下载适合你电脑系统的Node.js版本。Windows系统直接选择“Windows安装包（.msi）”的x64（如果你的电脑是64位系统的话）。

    然后打开安装的文件，一路摁Next（在第二个需要打勾同意用户协议）等待安装完成。

  • Git

    前往Git Download下载Git，然后安装。Windows系统直接在这里安装。

安装Hexo

  1. hexo的安装非常简单，首先在Powershell执行：

    1 npm install -g hexo

     

  2. 接下来找一个你喜欢的文件夹（必须是空的），进到文件夹中，摁住 Shift 然后右键空白区域，选择 在此处打开Powershell窗口 ，执行：
1 hexo init

创建仓库

  • 你需要一个仓库来存储博客网页，那就开始创建一个Github仓库吧~

    如果你没有Github账号，到这里去注册一个

    好，如果你注册完了或者登录好了，那么就来访问这里：创建新仓库

    把 RepoSitory Name 设为 【你的用户名】.github.io ，比如我叫waterblock79，那仓库名就应该是 waterblock79.github.io。

    填完之后就点击下面的 Create repository 创建仓库。

    接下来会跳转到仓库界面，复制并保存好这里的链接！

配置Hexo

  1. 现在在Powershell中执行：（别忘了替换引号里的内容~）

    1 git config --global user.name "你的Github用户名"
2 git config --global user.email "你注册Github使用的邮件地址"

举个例子，就像这样：

1 git config --global user.name "waterblock79"
2 git config --global user.email "waterblock79@outlook.com"

比如：

1 # Deployment
2 ## Docs: https://hexo.io/docs/deployment.html
3 deploy:
4   type: git
5   repo: https://github.com/waterblock79/waterblock79.github.io
6   branch: master

最后

（下面的命令需要在Hexo文件夹中执行）

（摁住 Shift 然后右键空白区域，选择 在此处打开Powershell窗口）

  1. 执行 hexo new 【文章名】 来新建文章，

    在你的博客文件夹的/source/_posts中编辑文章，可以使用Typora来编辑Markdown文件哦~

  2. 要上传文章？依次执行这三个命令即可

    1 hexo clean
2 hexo generate
3 hexo deploy

  1. （有可能会弹出一个Github的登录框，输入自己的Github账号邮箱和密码即可登录）

上传完了文章，就来看看效果吧，博客会在 【你的Github名称】.github.io 等着你。

  • 然后回到你刚刚的文件夹，打开文件夹中的 _config.yml ，找到最下面的 # Deployment，把内容改成：

    1 # Deployment
2 ## Docs: https://hexo.io/docs/deployment.html
3 deploy:
4   type: git
5   repo: 【你刚刚创建的仓库的网址】
6   branch: master

     上传完了文章，就来看看效果吧，博客会在 【你的Github名称】.github.io 等着你。
