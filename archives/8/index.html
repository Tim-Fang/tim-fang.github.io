Scratch辅助插件 Scratch 3 Developer Tools Browser Extension

本文最后更新于：1 个月前

Firefox(.xpi)Chrome(.crx)Edge(.crx)其他浏览器(.zip)

1.根据您的浏览器下载拓展

Chrome用户若显示程序包无效可下载zip版

2.安装

Firefox&Edge:

直接从拓展商店安装

Chrome&Others:

1.文件后缀改为zip 2.解压文件 3.打开浏览器拓展页 4.勾选开发者模式 5.加载已解压的拓展程序

*本拓展仅支持以Quantum&Chromium为内核的浏览器
